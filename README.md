# Enstack Agent OS — Landing Site

Marketing landing page for **Enstack Agent OS** by **enos** — the money stack for autonomous AI agents. Built from the product concept scope.

> Give your AI agents real money — and real control. A balance, a Visa card, and a bank account for every agent, so it can pay anyone, in any country, in seconds. One integration replaces five vendors.

## Stack

Plain static site — no build step, no dependencies.

- `index.html` — page structure and content
- `styles.css` — dark fintech theme, fully responsive
- `script.js` — sticky-nav state + reveal-on-scroll

## Run locally

Just open `index.html` in a browser, or serve the folder:

```bash
python -m http.server 8000
# then visit http://localhost:8000
```
